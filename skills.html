Compétences Acquises

À travers des exercices pratiques de DCLIC en HTML, CSS et JavaScript,j'ai développer une variété de compétences essentielles pour le développement web :

HTML (HyperText Markup Language)

  • Maîtrise de la structure des pages web grâce à des balises appropriées.
  • Création de formulaires, tableaux et listes, permettant de présenter des données de manière organisée.
  • Utilisation des attributs pour améliorer l'accessibilité et le référencement.

CSS (Cascading Style Sheets)

  • Application de styles pour améliorer l'apparence visuelle des pages.
  • Utilisation de sélecteurs, de propriétés et de valeurs pour manipuler la mise en page.

JavaScript

  • Ajout d'interactivité aux pages web, comme la gestion d'événements (clics, survols).
  • Manipulation du DOM (Document Object Model) pour modifier dynamiquement le contenu.
  • Validation de formulaires et gestion d'alertes, améliorant l'expérience utilisateur.

Ces compétences constituent une base solide pour moi qui a toujours souhaité m'engager dans une carrière dans le développement web ou améliorer mes connaissances en technologie.
